Quantities, Units, and Dimensions in Physics

1. Fundamental Quantities and Units

  • Length: \(L\) - meter (m)
  • Mass: \(M\) - kilogram (kg)
  • Time: \(T\) - second (s)
  • Electric Current: \(I\) - ampere (A)
  • Temperature: \(\Theta\) - kelvin (K)
  • Amount of Substance: \(N\) - mole (mol)
  • Luminous Intensity: \(J\) - candela (cd)

2. Derived Quantities and Units

  • Area: \(A = L^2\) - square meter (m2)
  • Volume: \(V = L^3\) - cubic meter (m3)
  • Speed/Velocity: \(v = \frac{L}{T}\) - meter per second (m/s)
  • Acceleration: \(a = \frac{L}{T^2}\) - meter per second squared (m/s2)
  • Force: \(F = \frac{ML}{T^2}\) - newton (N)
  • Pressure: \(P = \frac{M}{LT^2}\) - pascal (Pa)
  • Energy/Work: \(W = \frac{ML^2}{T^2}\) - joule (J)
  • Power: \(P = \frac{ML^2}{T^3}\) - watt (W)

3. Dimensional Formulas

  • Force: \([F] = [M][L][T^{-2}]\)
  • Work/Energy: \([W] = [M][L^2][T^{-2}]\)
  • Power: \([P] = [M][L^2][T^{-3}]\)
  • Pressure: \([P] = [M][L^{-1}][T^{-2}]\)
  • Velocity: \([v] = [L][T^{-1}]\)
  • Acceleration: \([a] = [L][T^{-2}]\)
  • Density: \([\rho] = [M][L^{-3}]\)
  • Momentum: \([p] = [M][L][T^{-1}]\)
