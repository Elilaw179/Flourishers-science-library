Scalars and Vectors in Physics

1. Scalars and Vectors

Scalar Quantity: A physical quantity that has only magnitude and no direction.

  • Examples: Mass (m), Temperature (T), Time (t), Distance (d), Speed (v)

Vector Quantity: A physical quantity that has both magnitude and direction.

  • Examples: Displacement (s), Velocity (v), Acceleration (a), Force (F)

2. Representation of Vectors

Vectors can be represented in two main ways:

2.1 Graphical Representation

Graphically, vectors are depicted as arrows where:

  • The length of the arrow represents the magnitude of the vector.
  • The direction of the arrow indicates the direction of the vector.

2.2 Component Form

Vectors can be broken down into their components along the x, y, and z axes.

Example: A vector A in 2D can be represented as:

A = Axi + Ayj

Where:

  • Ax = Component of A along the x-axis
  • Ay = Component of A along the y-axis
  • i, j = Unit vectors in the x and y directions respectively

3. Magnitude of a Vector

The magnitude (length) of a vector can be calculated using the Pythagorean theorem.

For a 2D Vector:

|A| = √(Ax² + Ay²)

For a 3D Vector:

|A| = √(Ax² + Ay² + Az²)

4. Unit Vectors

A unit vector has a magnitude of 1 and indicates direction.

Notation: î, ĵ, k̂ for the x, y, and z axes respectively.

Example: To find the unit vector of vector A:

â = A / |A|

5. Vector Addition and Subtraction

5.1 Graphical Method

Vectors are added using the tip-to-tail method:

  • Place the tail of the second vector at the tip of the first vector.
  • The resultant vector (A + B) is drawn from the tail of the first vector to the tip of the second vector.

5.2 Component Method

Vectors are added component-wise:

A + B = (Ax + Bx)i + (Ay + By)j + (Az + Bz)k

Similarly, for subtraction:

A - B = (Ax - Bx)i + (Ay - By)j + (Az - Bz)k

6. Scalar Multiplication

Multiplying a vector by a scalar changes its magnitude but not its direction (unless the scalar is negative, which also reverses the direction).

Formula: If k is a scalar, then:

kA = (kAx)i + (kAy)j + (kAz)k

7. Dot Product (Scalar Product)

The dot product of two vectors results in a scalar.

Formula:

A · B = |A| |B| cos(θ)

Where θ is the angle between vectors A and B.

Component Form:

A · B = AxBx + AyBy + AzBz

8. Cross Product (Vector Product)

The cross product of two vectors results in a vector that is perpendicular to both original vectors.

Formula:

A × B = |A| |B| sin(θ) n

Where:

  • θ is the angle between A and B
  • n is a unit vector perpendicular to the plane containing A and B

Component Form:

A × B = (AyBz - AzBy)i - (AxBz - AzBx)j + (AxBy - AyBx)k

9. Projection of Vectors

The projection of vector A onto vector B is the component of A in the direction of B.

Formula:

projBA = (A · B / |B|²) B

10. Angle Between Two Vectors

The angle θ between two vectors can be found using the dot product formula:

cos(θ) = (A · B) / (|A| |B|)

Solving for θ:

θ = cos-1((A · B) / (|A| |B|))

11. Work Done by a Force

Work is a scalar quantity that measures the energy transfer when a force is applied over a displacement.

Formula:

W = F · d = |F| |d| cos(θ)

Where:

  • W = Work done (in joules, J)
  • F = Force vector (in newtons, N)
  • d = Displacement vector (in meters, m)
  • θ = Angle between the force and displacement vectors

12. Resolving Vectors into Components

Any vector can be resolved into perpendicular components using trigonometric functions.

For a vector A making an angle θ with the x-axis:

Ax = |A| cos(θ)

Ay = |A| sin(θ)

Az = |A| sin(φ) (if applicable, where φ is the angle with the z-axis)

13. Vector Magnitude from Components

Given the components of a vector, its magnitude can be calculated as:

|A| = √(Ax² + Ay² + Az²)

14. Vector Addition Using Components

To add two vectors using their components:

A + B = (Ax + Bx)i + (Ay + By)j + (Az + Bz)k

15. Scalar Triple Product (Advanced)

The scalar triple product of three vectors A, B, and C is defined as:

A · (B × C)

This represents the volume of the parallelepiped formed by the three vectors.

Formula:

A · (B × C) = Ax(ByCz - BzCy) - Ay(BxCz - BzCx) + Az(BxCy - ByCx)

16. Vector Projection Using Components

When projecting vector A onto vector B using components:

projBA = [(AxBx + AyBy + AzBz) / (Bx² + By² + Bz²)] B

17. Orthogonal Vectors

Two vectors are orthogonal (perpendicular) if their dot product is zero.

A · B = 0

18. Parallel Vectors

Two vectors are parallel if one is a scalar multiple of the other.

A = kB

Where k is a scalar.

19. Resultant Vector

The resultant vector is the single vector that represents the combined effect of two or more vectors.

Resultant (R) = A + B + C + ...

20. Vector Decomposition

Any vector can be decomposed into its components along mutually perpendicular axes.

Example: Decomposing vector A into x and y components.

A = Axi + Ayj
